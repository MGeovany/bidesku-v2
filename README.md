## 🚀 Get Started

You need to be using:

- [Git](https://git-scm.com/downloads)

- [Nodejs v16 or higher](https://nodejs.org/es/download/)

Clone the repository & install dependencies:

```bash

git clone https://github.com/eriandev/bidesku-v2

cd bidesku-v2

npm install

```
